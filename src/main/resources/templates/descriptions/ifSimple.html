The \{#if} section is a basic control flow construct. The simplest possible version accepts a single parameter and renders the content if the condition is not falsy, i.e. if the value is not null, false, an empty collection, an empty map, an empty array, an empty string/char sequence or a number equal to zero. However, you can also use:
  • operators, e.g.: \{#if item.age > 10},
  • multiple conditions, e.g.: \{#if item.age > 10 && item.price > 500},
  • overridde the precedence rules by parentheses, e.g.: \{#if (item.age > 10 || item.price > 500) && user.loggedIn},
  • add any number of \{#else} blocks.
