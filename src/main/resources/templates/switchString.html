{#include base} {#title}Switch w/ String{/title} {#description}switch is an alias for the when section that matches a tested value against all blocks sequentially until a condition is satisfied. The first matching block is executed. All other blocks are ignored.{/} {#labels} expression when section {/labels} {#content} 
{|<p> Hello 
{#switch name}
   {#case 'Edgar'}
      man
   {#case in 'John' 'Foo'}
      brother
   {#else}
      there
{/switch}! </p>|}
